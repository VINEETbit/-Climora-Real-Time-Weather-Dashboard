🌤 About Climora

Climora is a real-time weather dashboard that provides accurate temperature, humidity, and wind speed data for cities worldwide. It uses live weather APIs to ensure up-to-date results.

📖 How to Use Climora
  • 🔍 Enter any city name in the search bar and click Search.
  • 🌡 View real-time temperature, humidity, and wind speed.
  • 📊 Check common cities in the weather table below.
  • 📍 Use the Quick Cities dropdown for instant results.
  • 🏠 Click Home to reset the dashboard.
🚀 Features
  • ✔ Real-time weather data
  • ✔ Global city search
  • ✔ Responsive mobile design
  • ✔ Clean and modern UI

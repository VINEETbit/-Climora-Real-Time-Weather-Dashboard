# -Climora-Real-Time-Weather-Dashboard

>Climora is a modern real-time weather web application that allows users to search weather information for any city worldwide. It provides temperature, humidity, wind speed, maximum temperature, and minimum temperature using live weather APIs.


🚀 Features

🌍 Search weather for any city worldwide

🌡 Real-time temperature display

🔺 Maximum temperature

🔻 Minimum temperature

💧 Humidity level

💨 Wind speed

🌤 Dynamic weather icons

📊 Weather table for common cities

📱 Fully responsive design

🎨 Modern animated UI

☁ Weather-themed visuals


🛠 Technologies Used
>Frontend

>HTML5

>CSS3

>JavaScript (ES6)

>Frameworks / Libraries

>Bootstrap 5

>API

>Open-Meteo API

>Open-Meteo Geocoding API

These APIs provide real-time weather data and city coordinates.

⚙️ How It Works

>User enters a city name in the search bar.

>The app sends a request to the Geocoding API to get latitude and longitude.

>Using those coordinates, it requests weather data from the Open-Meteo API.

>The weather data is displayed on the dashboard cards.
